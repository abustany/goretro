Design ideas

We want to play on the "retro" name, but keep things readable. The color palette is roughly inspired from the original NES colors.

Fonts

The logo font is Press Start 2P, we don't use it anywhere else because it gets tiring quickly. The rest of the app uses whatever sans-serif gives us, although one could probably spend time finding something better.
